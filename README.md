#  JournalApp

A secure journal management web application built using **Spring Boot**. This app allows users to write journal entries, analyze sentiments, and receive personalized feedback.
---

##  Tech Stack

- **Backend:** Java, Spring Boot
- **Security:** Spring Security, JWT
- **Auth:** Google OAuth 2.0
- **Database:** JPA, Hibernate (MySQL)
- **Caching:** Redis
- **Scheduler:** Spring Scheduled Tasks
- **API Docs:** Swagger
- **Build Tool:** Maven
